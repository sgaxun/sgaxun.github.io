概述

  • JVM：Java Virtual Machine，也就是 Java 虚拟机
  • 所谓虚拟机是指：通过软件模拟的具有完整硬件系统功能的、运行在一个完全隔离环境中的计算机系统
  • JVM 是通过软件来模拟 Java 字节码的指令集，是 Java 程序的运行环境
